Protocol Handler Setup (Optional)

Make acestream:// links open automatically in this interface:

  1. Bookmark this page in your browser
  2. Configure your browser:
    • Chrome/Edge: Settings → Advanced → Site settings → Handler → Add this site
    • Firefox: about:config → browser.link.open-newwindow.restriction = 0
    • Safari: Preferences → Websites → Custom Settings
  3. Or install a browser extension: "AceStream Link Handler"
